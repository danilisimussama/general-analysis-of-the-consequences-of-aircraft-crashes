## General alysis of the consequences of aircraft crashes in the 20th and 21st centuries
Это аналитическое исследование последствий авиационных происшествий на основе открытой базы данных по авариям и жертвам. 
Цель проекта — оценить распределения смертности и числа пострадавших по категориям воздушных судов, сравнить ключевые метрики между категориями и представить результаты в виде таблиц и графиков.
